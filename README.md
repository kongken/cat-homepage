# 🐱 LoveCat Tools

Collection of handy online tools for developers, with great UX.

## Tools

- **Tools Collection** - https://tools.lovec.at/  
  Collection of handy online tools for developers

- **Subtitle Translator** - https://subtitle-translator.lovec.at/  
  Blazing-fast batch subtitle translation for SRT/ASS/VTT/LRC — 70+ languages, AI-powered 批量字幕翻译

## Development

### Install Dependencies

```bash
yarn install
```

### Start Development Server

```bash
yarn dev
```

### Build for Production

```bash
yarn build
```

## Tech Stack

- Vue 3
- Vuetify 3
- Vite
